搭建个人博客 Github&Hexo(配置)

参考链接：https://www.jianshu.com/p/b03b39d5a4ef

说明：

主要是记录从头开始搭建个人blog时遇到的错误和解决方法

一、安装Hexo

  • 安装（打开git bash）
npm install hexo-cli -g
  • 进入放博客的文件夹，执行下列的命令
hexo init blog    //该命令会创建一个blog的文件夹，里面为相关的配置文件
cd blog           //进入文件夹
npm install       //安装package.json里配置的模块
hexo server       //运行本地服务，在网页中查看效果
  • 安装完成后进入文件夹，可以看到如下文件夹及文件：
node_modules npm的安装模块
scaffolds 布局文件
source 资源文件，文件都放在该文件下的_posts
themes 主题文件
.gitignore git上传忽略文件配置
_config.yml 核心配置文件
package.json 描述npm安装包的文件

二、github上传配置

  • 说明：当在本地写好文章后就要传到github上，下面做相关配置。
#打开_config.yml文件，在最小面找到deploy字段，做如下配置：
deploy:
type: git
repo: git@github.com:yourname/yourname.github.io.git(将yourname换成你的GitHub的名字)
branch: master

#然后执行下代码，安装相关模块，防止上传代码出错。
npm install hexo-deployer-git --save
  • 配置GitHub

相关信息查看：<传送门>

三、生成文件上传github

  • 生成静态页面文件夹public

    hexo generate

  • 开启本地服务查看效果（默认端口4000）

    hexo server

  • 将.deploy目录部署到GitHub

    hexo deploy

  • 删除public文件夹

    hexo clean

  • tip:每次上传文件前要先删除public文件夹，在生成静态页面，最后上传github，故执行如下命令

hexo clean
hexo generate
hexo deploy
  • tip:每次运行时都要执行三个命令会很麻烦，可以在package.json文件里加入scripts字段：
"scripts": {

    "dev": "hexo clean && hexo g && hexo d"

}
# 然后再git bash里执行npm run dev 即可

四、错误纠正

  • 运行npm run dev 可能会出现错误：
npm ERR! code EJSONPARSE      #说明json格式错误
npm ERR! file F:\\Github_desk\\blog\\blog\\package.json
npm ERR! JSON.parse Failed to parse json
npm ERR! JSON.parse Unexpected string in JSON at position 655 while parsing '{
npm ERR! JSON.parse   "name": "hexo-site",
npm ERR! JSON.parse   "version": "0'
npm ERR! JSON.parse Failed to parse package.json data.
npm ERR! JSON.parse package.json must be actual JSON, not just JavaScript.

npm ERR! A complete log of this run can be found in:
npm ERR!     E:\\New Folder\\node_cache\\_logs\\2022-01-12T05_03_59_255Z-debug.log
  • 打开package.yml
{
  "name": "hexo-site",
  "version": "0.0.0",
  "private": "true",  #这里将true加上双引号
  "scripts": {
    "build": "hexo generate",
    "clean": "hexo clean",
    "deploy": "hexo deploy",
    "server": "hexo server"
  },
  "hexo": {
    "version": "5.4.0"
  },
  "dependencies": {
    "hexo": "^5.0.0",
    "hexo-deployer-git": "^3.0.0",
    "hexo-generator-archive": "^1.0.0",
    "hexo-generator-category": "^1.0.0",
    "hexo-generator-index": "^2.0.0",
    "hexo-generator-tag": "^1.0.0",
    "hexo-renderer-ejs": "^1.0.0",
    "hexo-renderer-marked": "^4.0.0",
    "hexo-renderer-stylus": "^2.0.0",
    "hexo-server": "^2.0.0",
    "hexo-theme-landscape": "^0.0.3"
  },               # 末尾这里要记得加上逗号
  "scripts":{
    "dev": "hexo clean %% hexo g && hexo d"
  }
}
